Программа запускает локальные html-страницы,
с которых можно сделать запрос к hh.ru и посмотреть его результаты.

Запускать надо модуль 'main.py'

Назначения файлов:

main.py - основной модуль, запускает Flask

activate_regions.py - создает словари макрорегионов (субъектов РФ)
                                    и городов

db_exchange.py - функции ввода-вывода в базу данных

db_new_activate.py - создает заново базу данных
при начале работы и заполняет таблицы статичных данных (регионы и города)

db_obj_declaration.py - Объявление классов для базы данных

req_to_hh.py - запрос к api hh.ru
записывает последние результаты в таблицу  Vacancy базы данных

в папке templates находятся html-файлы и bootstrap
