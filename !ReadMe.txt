Программа запускает локальные html-страницы,
с которых можно сделать запрос к hh.ru и посмотреть его результаты.

Запускать надо модуль 'main.py'

Назначения файлов:

main.py - основной модуль, запускает Flask

activate_data.py - читает из файла data/regions.json
наименования и коды регионов для использования в меню запроса

req_to_hh.py - запрос к api hh.ru
записывает последние результаты в data/vac_lst.json

в папке templates находятся html-файлы и bootstrap

для иллюстрации - копии экрана: Screens_11_08_2022.pdf
